# authweb
A Minecraft Yggdrasil Web Server of Node.js

## 介绍

基于 `Node.js` 和 `MySQL` 的 Minecraft `Yggdrasil`  服务器. 待添加...

## 部署

#### 要求

- Node.js
- MySQL
- Git

运行:

```sh
git clone https://github.com/McMoonLakeDevAuth/authweb.git
cd authweb
npm install
```

接下来:

- 配置 config.js
- 本地化 local.json

运行:

```sh
npm start
```

开始访问你的 `authweb` 服务器.
